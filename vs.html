koa VS express

Koa VS Express

Koa

  • 支援 async & await
  • 可自由挑選各種組件
const Koa = require('koa')
const app = new Koa()
const hostname = '127.0.0.1'
const port = 3000

app.use(async (ctx) => {
  ctx.body = 'Hello World'
})
app.listen(port, hostname, () => console.log(`Example app listening on port ${port}!`))

Express

https://expressjs.com/

  • 有預設的 router 模組
  • 有預設的 body parse 模組
const express = require('express')
const app = express()
const hostname = '127.0.0.1'
const port = 3000

app.get('/', (req, res) => {
  res.send('Hello World'))
}
app.listen(port, hostname, () => console.log(`Example app listening on port ${port}!`))
